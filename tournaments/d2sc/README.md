# Dota 2 Star Championship (2011) - Ultimate Recovery Report

**Status**: Playoffs Fully Recovered (14/14 matches!) 🏆 | Group Stage: In Progress
**Dates**: Dec 8-11, 2011
**Location**: Kiev Cyber Sports Arena, Ukraine
**Prize Pool**: $20,000 USD
**Organizer**: Star Ladder

## 🏆 Tournament Overview

One of the very first Dota 2 LAN tournaments, held in Kiev. It featured top teams like **Natus Vincere**, **Moscow Five**, and **Mousesports**.

### Final Standings
