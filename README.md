# Dota Stats Archive

Archived Dota 2 tournament statistics and match data.

## About

This repository contains recovered tournament data, match statistics, and historical records from various Dota 2 competitive events.

**🤖 Created with the help of LLM** — data recovery, structuring, and documentation assisted by AI.

## Structure

```
├── tournaments/     # Tournament metadata and brackets
├── matches/         # Individual match data
├── players/         # Player statistics and profiles
├── teams/           # Team rosters and history
└── raw/             # Raw recovered data (pre-processing)
```

## Data Format

All structured data is stored in JSON format for easy parsing and analysis.
